# ⚙️ Running the Project

## • Dev mode

1. Enter the `app` folder:

```bash
cd app
```

2. Install dependencies:

```bash
$ npm i
```

3. Run the app:

```bash
$ npm start
```

4. Access the URL: http://localhost:3000
